SoftBox Project

ALX Software Engineering Foundation Graduation Project

About SoftBox

SoftBox is an application implemented entirely using C programming language and Linux kernel's containerization features. It creates containers using cgroups, Linux namespaces, and networking to run Docker images.

Key Features

  • Utilizes Linux kernel's containerization capabilities
  • Implements cgroups for resource management
  • Uses Linux namespaces for isolation
  • Supports running Docker images
  • Developed as part of ALX Software Engineering Foundation program

Technical Details

SoftBox is built using:

  • C Programming Language
  • cgroups v2
  • Linux Namespaces
  • Networking linux kernel features for container communication

Requirements

Ubuntu (Linux distribution) with cgroups v2 support
